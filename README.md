# MercadoMotor

Plataforma premium para la compra y venta de vehículos.

## Tecnologías
- **Frontend**: Next.js 15 (App Router)
- **Base de Datos**: Prisma con SQLite/Supabase
- **Estilos**: Vanilla CSS con variables de diseño premium
- **Autenticación**: NextAuth.js

## Desarrollo
1. `npm install`
2. `npx prisma generate`
3. `npm run dev`

## Rebranding
Este proyecto fue migrado de la identidad AutoVirtual a MercadoMotor en Mayo de 2026.
